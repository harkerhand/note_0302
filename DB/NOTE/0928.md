# 数据模型

描述数据的概念和工具：

- 结构
- 操作
- 约束

  

# 关系数据模型

- 结构：关系（表格）
- 操作：代数操作
- 约束：主键、外键等等

逻辑模式：表格及表格间的关系

物理模式：B+树

  

关系（表）是笛卡尔积的子集
